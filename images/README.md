# Portrait Images

Place your 4 character portrait images in this folder with these exact filenames:

| Filename         | Emotion shown         |
|------------------|-----------------------|
| `friendly.png`   | Smiling / friendly    |
| `excited.png`    | Happy / excited       |
| `sad.png`        | Sad / sulking         |
| `questioning.png`| Surprised / questioning |

The game will automatically display the correct portrait for each scene.
If an image is missing, an emoji fallback will be shown instead.
